# Social Media Skill

## Overview
Create and manage social media content across platforms with scheduling and engagement strategies.

## Commands

### generate-posts
Create platform-specific social media posts with appropriate formatting and hashtags.

**Parameters:**
- `platform` - Target platform (twitter, linkedin, instagram, facebook, tiktok)
- `topic` - Post topic or theme
- `count` - Number of post variations

### plan-campaign
Design a multi-platform social media campaign with coordinated messaging and timelines.

**Parameters:**
- `objective` - Campaign goal (awareness, engagement, conversion, launch)
- `platforms` - Target platforms
- `duration` - Campaign duration
- `budget` - Budget tier (organic, low, medium, high)

### analyze-engagement
Analyze social media engagement patterns and provide optimization recommendations.

**Parameters:**
- `platform` - Platform to analyze
- `metrics` - Key metrics to focus on (reach, engagement rate, clicks, conversions)
- `period` - Analysis time period
